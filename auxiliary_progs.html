These auxiliary programs are installed as part of phydms. All of them can be run from the command line. They aid in running multiple analyses with the phydms program, or in analyzing and visualizing the results.

  • the phydms_comprehensive program
  • the phydms_plotselection program
  • the phydms_analyzeselection program
  • the phydms_renumber program
